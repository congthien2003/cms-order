# BrewOrder — Landing Page

Landing page for the **BrewOrder** POS system — a complete point-of-sale solution for beverage shops.

Built with [Next.js](https://nextjs.org) and [TailwindCSS](https://tailwindcss.com).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

Fonts are loaded via Google Fonts (`DM Sans` and `Playfair Display`) using a `<link>` tag in `layout.tsx`.

## Sections

- **Hero** — Headline, subheading, CTA buttons and feature stats
- **Features** — 6 core feature cards (Order Management, Menu Builder, Vouchers, Dashboard, Mobile App, Real-time Queue)
- **How It Works** — 4-step order workflow
- **Tech Stack** — Backend, Admin Panel, Mobile App, Infrastructure
- **Interface Preview** — Mock admin dashboard UI
- **CTA** — Call-to-action with GitHub link

## Build

```bash
npm run build
npm start
```
