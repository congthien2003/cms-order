# Store Order App

## 1. Clone
```
git clone https://gitlab.com/sabo-techspace/cms-and-ordering/store-order-app
```

## 2. Change ConnectionString in API
```
    "ConnectionStrings": {
      "sqlConnection": "server=localhost,6001; database=CMS_Management; Trusted_Connection=True; User Id=sa; Password=Password123@;TrustServerCertificate=True;"
    },
```

## 3. Install package
```
npm install
```
## 4. Run

```
npm run dev
```
- Client will run in port 3000:
- API will run in port 5001
